# First Step:
1. Git clone the project

## Front-End:


1. cd frontend
2. npm install
3. npm run dev

## Back-End:

1. cd backend
2. npm install
3. Use .env.example to complete the informations of your enviroment and rename .env 
4. npm run typeorm migration:run -- -d ./src/data-source
5. npm run dev

The docs can be found at "/api-documentation"

Happy coding!
